Destination Formats

Google Maps URL — paste the full URL. For short links (maps.app.goo.gl), open the link in a browser first and copy the full URL from the address bar.
Apple Maps URL — paste a maps.apple.com link that contains coordinates. For shared links without coordinates, open the link in a browser first and copy the full URL.
Plus Code — 87G8Q2PQ+VX (full codes only). plus.codes/map is a great way to find coordinates and Plus Codes.
Cast a Point
Swipe up 3 times over the compass to enter cast mode. Aim your phone in a direction, drag to pick a distance, then tap Cast to set a waypoint.
Install for Offline Use
In Chrome, tap the three-dot menu and select “Add to Home screen.” The app will work fully offline once installed.
